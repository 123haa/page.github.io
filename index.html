🔹 前提准备

  • 一个 GitHub 账号（免费注册）。
  • 你本地已经有一个网页文件，比如 index.html。

🚀 步骤 1：新建仓库

  1. 登录 GitHub
  2. 右上角点 “+” → New repository。
  3. 输入仓库名（推荐 my-website 或 username.github.io）。
  4. 如果你想要个性主页，必须用格式：你的用户名.github.io。
  5. 例：用户名是 xiaoming → 仓库名必须是 xiaoming.github.io。
  6. 选择 Public（公开）。
  7. 点击 Create repository。

🚀 步骤 2：上传网页文件

  1. 在仓库页面里，点 “Add file” → “Upload files”。
  2. 上传你的网页文件：至少要有一个 index.html（这是网站默认入口）。
  3. 其他的 CSS、图片文件一并上传。
  4. 点 Commit changes(提交更改) 保存。

🚀 步骤 3：开启 GitHub Pages

  1. 打开仓库 → 点上方的 Settings。
  2. 左边菜单找到 Pages。
  3. 在 Build and deployment → Source 里，选择：Deploy from branch
  4. 分支选 main，文件夹选 /(root)。
  5. 点 Save。

🚀 步骤 4：等待部署

GitHub 会自动帮你部署，大概需要 1-2 分钟。
部署成功后，页面会显示一个网址，比如：
https://你的用户名.github.io/
打开网址，就能看到你上传的网页啦！ 🎉

🚀 步骤 5（可选）：绑定自己的域名

如果你买了自己的域名，可以在 Pages 设置里添加自定义域名。
GitHub 会自动配置免费 HTTPS。

✅ 小结

GitHub Pages 免费、永久有效，非常适合小白建个人主页、说明文档、静态网站。
只要你更新仓库内容，网站就会自动刷新。
